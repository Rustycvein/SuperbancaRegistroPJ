# Superbancas_Registro

## Descripción
**Superbancas_Registro** es un prototipo de página de registro para nuevos clientes de la empresa **Superbancas**.  
El objetivo del proyecto es digitalizar el proceso de registro que normalmente realiza el personal de servicio al cliente de forma manual, permitiendo que los clientes completen un formulario con la información requerida de manera más rápida, organizada y eficiente.

Este sistema busca reducir errores, optimizar el tiempo de atención y facilitar la gestión de los datos de los clientes.

---

## Tecnologías utilizadas
- TypeScript
- HTML
- CSS
- Node.js
- Express.js
- SQLite

---

## Librerías y dependencias
El proyecto utiliza las siguientes dependencias:

- **express**: creación del servidor backend
- **cors**: manejo de políticas CORS
- **multer**: manejo de subida de archivos
- **sqlite3 / sqlite**: base de datos local
- **exceljs**: generación y exportación de datos a archivos Excel
- **nodemailer**: envío de correos electrónicos
- **fs**: manejo del sistema de archivos
